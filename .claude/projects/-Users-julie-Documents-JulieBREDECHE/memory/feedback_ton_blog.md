Pas de tutoiement dans les articles de blog. Utiliser "vous" ou des formulations impersonnelles.

**Why:** Julie trouve le tutoiement trop informel pour son positionnement de consultante experte.

**How to apply:** Dans tous les articles de blog, bannir "tu/te/ton/ta/tes". Préférer le "vous", les formulations à l'infinitif, les phrases nominales, ou les tournures impersonnelles ("l'app ne voit que", "la délégation échoue", etc.). Rester concis, pas de familiarité excessive.
