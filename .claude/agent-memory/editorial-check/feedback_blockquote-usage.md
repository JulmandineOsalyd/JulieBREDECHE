Le blockquote Markdown (`>`) est réservé aux verdicts forts dans la section `## Ma conclusion`. Ne jamais l'utiliser pour du contenu informatif (prompt technique, définition, info secondaire).

**Why:** La charte éditoriale de Julie réserve le blockquote au « martèlement d'un verdict fort ». Utiliser le blockquote pour un prompt IA crée une confusion de hiérarchie visuelle.

**How to apply:** Quand un blockquote contient un prompt, du code ou une information technique, le convertir en bloc de code triple backticks ou en prose. Vérifier à chaque audit que le seul blockquote restant est celui du verdict final dans `## Ma conclusion`.
